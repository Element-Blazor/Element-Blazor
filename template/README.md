# Template Workspace

This folder hosts the Element style enterprise template submodule.

Expected submodule:

- Path: template/ElementBlazorTemplate
- URL: https://github.com/maikebing/ElementBlazorTemplate

Initialize/update:

```powershell
git submodule sync --recursive
git submodule update --init --recursive
```

Template goal:

- Element style + Chinese enterprise admin dashboard
- ASP.NET Core DDD architecture
- EF Core with optional PostgreSQL / SQLite / SQL Server
- Element-Blazor as out-of-box frontend UI
